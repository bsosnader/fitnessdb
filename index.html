Fitness Attendance

Notice: As of Summer 2018, Penn State has taken down the website I used to scrape this data. :(

Below you'll find a comparison between the capacities of the three Penn State gyms over a period of several hours. Information is updated every ten minutes. If you'd like to read more about how this was created, click here!

Technologies Used
Python

Python is used to scrape data from the original page, format it, and insert it into a database. Special shout out to Selenium and BeautifulSoup.

InfluxDB

InfluxDB is a robust time series database which stores the scraped values in ten minute intervals.

ChartJS and Skeleton

The data is manipulated and visualized by ChartJS, a flexibile charting library for javascript. Styles provided by Skeleton

Nginx and Let's Encrypt

The site is served using nginx with a reverse proxy pointing toward the database. It's secured with a free TLS Certificate from the wonderful people at Let's Encrypt
